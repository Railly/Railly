Effective agent design boils down to context management. Context is a finite resource with diminishing returns - like human working memory.

A higher capacity model with caching can be cheaper than a lower cost model without it. Progressive disclosure: don't load all tool definitions upfront. Successful agents use surprisingly few tools (~12 for Claude Code, &lt;20 for Manus).
