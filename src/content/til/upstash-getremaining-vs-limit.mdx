---
title: "Upstash getRemaining() vs limit()"
pubDate: 2026-02-19
topic: Infrastructure
link: https://upstash.com/docs/oss/sdks/ts/ratelimit/methods#getremaining
---

`limit()` consumes a token. `getRemaining()` doesn't. Use `getRemaining()` for status endpoints and UI counters.

```ts
// read-only, no token consumed
const { remaining } = await rl.getRemaining(id);

// consumes a token
const { success } = await rl.limit(id);
```
