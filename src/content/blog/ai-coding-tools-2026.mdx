## The principle before the tools

An AI for coding lives in three places:

1. **The chat**: to think, explain, plan.
2. **The IDE**: to autocomplete and edit files with context.
3. **The CLI**: to run real tasks on your repo (commands, file edits, commits).

You don't need the best tool in each layer. You need **a decent one in each** and for them to complement each other.

## The engine: use the Gemini Pro your tuition already paid for

If your university gives you Google AI Pro / Gemini Advanced, that's your most underrated asset. Gemini 3.x today is on par with frontier models for code and reasoning, and it ships a huge context window: you can paste an entire repo and ask questions. For a student, that's gold.

Use it like your "Claude": planning, pasting long error logs, asking for explanations, brainstorming architecture.

**Free complement:** ChatGPT free for a second opinion. Having two different models on hand trains your eye to spot when one is hallucinating.

## In the terminal: Gemini CLI

Here's the gem nobody uses: **Gemini CLI** is free, open source, and gives you ~1,000 requests a day on a personal Google account. It's the direct equivalent of Claude Code, no card required.

```bash
npx https://github.com/google-gemini/gemini-cli
```

Open it inside your repo and talk to it like a coworker: "read this project and tell me what it does", "add tests to this file", "fix this bug". It reads files, runs commands, edits code.

**Alternative:** **OpenAI Codex CLI** is also open source, and now free with any ChatGPT account (yes, even free tier). 30-second setup if you already have OpenAI.

My advice: install both. They're free. You'll feel which one matches your brain.

## In the editor: Cursor or VS Code + Copilot

Two honest paths:

**Path A: VS Code + GitHub Copilot (free for students).** If you have the GitHub Student Pack, Copilot Pro is **free** while you're a student. This is the closest thing to "free magic" right now. Solid autocomplete, integrated chat, agent mode. Done.

**Path B: Cursor free tier.** The editor I use. The free tier is enough to learn what programming with an agent at your side feels like. When the credits run out, you go back to VS Code + Copilot and lose nothing.

What I do **not** recommend up front: paying for Cursor Pro as a student. Squeeze the free tier first.

## What about the new wave: Kimi, Z.ai, MiniMax?

If you've heard of [Kimi K2](https://kimi.moonshot.cn), [Z.ai's GLM](https://chat.z.ai), or [MiniMax](https://chat.minimax.io), here's the short version: these are top-tier Chinese models with **free web chats** and absurdly cheap APIs. MiniMax was the #1 model by volume on OpenRouter in February 2026. Kimi has a 256K context window. They're real options.

Best way to use them on a budget: free web chat for daily reasoning, or plug them into an open-source CLI (Aider, opencode, Crush) via [OpenRouter](https://openrouter.ai), which lets you swap models with a single config line.

## The middle layer nobody mentions: AI gateways

When you start paying per token, you'll want one URL in front of every provider so you can swap models without rewriting code. Two free options:

- **[Vercel AI Gateway](https://vercel.com/docs/ai-gateway)**: $5/mo in credits, zero markup, BYOK. Plays perfect with the AI SDK.
- **[Cloudflare AI Gateway](https://developers.cloudflare.com/ai-gateway)**: free on every Cloudflare plan, adds caching, rate limits, and observability.

These aren't models. They're the *fabric* in front of models. Once you're shipping anything with an LLM in production, you want one of these in the path.

## The stack I'd build today, from zero

- **Think / plan:** Gemini 3.x (university account) + ChatGPT free as second opinion.
- **Editor:** VS Code + Copilot via GitHub Student Pack.
- **Terminal:** Gemini CLI as primary, Codex CLI as backup.
- **Cost:** $0.

That gives you 80% of what I have paying multiple subscriptions. The real difference isn't the tool. It's how many hours you spend learning to use it well.

## One warning I wish I'd been given

Don't use AI to *avoid* thinking. Use it to *accelerate* the thinking you already did. The trap when you're starting is that it's easy to become a copy-paster who doesn't understand their own code. When something works, stop and ask yourself: *could I rewrite this without the AI?* If the answer is no, go back and read it line by line until the answer is yes.

The tools are free. Understanding isn't downloadable.
